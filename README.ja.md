# instasplitter

Instagram用にJPGファイルを複数の画像に分割するシンプルなWebアプリケーションです。

## デモ
[https://code4fukui.github.io/instasplitter/](https://code4fukui.github.io/instasplitter/)

## 機能
- Instagram投稿用にJPGファイルを1〜9枚の画像に分割
- Instagram向けに画像サイズとレイアウトを最適化
- モバイルとデスクトップの両方のレイアウトに対応
- 分割する行数と列数を選択可能
- 分割した画像のダウンロード機能を提供

## 使い方
1. 指定されたエリアにJPGファイルをドラッグ＆ドロップします。
2. 分割する画像の行数と列数を選択します。
3. モバイル端末向けにレイアウトを最適化する場合は、「スマホ画面用」オプションにチェックを入れます。
4. ダウンロードボタンをクリックして、分割された画像をZIPファイルとして保存します。
