# instasplitter

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple web application that splits JPG files into multiple images for Instagram.

## Demo
[https://code4fukui.github.io/instasplitter/](https://code4fukui.github.io/instasplitter/)

## Features
- Splits JPG files into 1-9 images for Instagram posting
- Optimizes image size and layout for Instagram
- Supports both mobile and desktop layouts
- Allows selecting the number of rows and columns
- Provides a download option for the split images

## Usage
1. Drag and drop a JPG file onto the designated area.
2. Select the number of rows and columns for the split images.
3. Check the "Mobile screen" option to optimize the layout for mobile devices.
4. Click the download button to save the split images as a ZIP file.
